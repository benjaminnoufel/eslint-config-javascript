# Contributing

## Requirements

- [Git][git]
- [Docker][docker]
- [Docker Compose][docker-compose]
- [GNU/Make][make]

## Installation

```console
$ git clone https://github.com/benjaminnoufel/eslint-config-javascript.git
```
## Build

```console
$ make build
```

## Clean

```console
$ make clean
```
